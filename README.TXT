Требования:
~~~~~~~~~~

1. Сервер: Windows, Linux.
2. Веб-сервер: Apache (или IIS c PHP).
3. PHP 4 или выше.
4. MySQL 3 или выше.


Установка:
~~~~~~~~~~

1. Заливаем на сервер.

2. Выставляем права на запись (0777 - для *nix) на папки:
	/upload
	/heart/cache
	/heart/tiny_mce/plugins/ajaxfilemanager/session

3. Создаем базу данных с именем "globalstar" или другую.

4. Открываем файл "/heart/inc/dbconfig.php" и указываем настройки к БД (базу, пользователь, пароль).

5. Выполняем импорт данных в БД с файла "/globalstar.sql".

6. Заходим в админку и наполняем контент. По умолчанию логин:пароль = admin:admin

7. Юзаем сайт.


Необязательно:
Если на сайт большая нагрузка, можна установить кеширование страниц. Для этого открываем файл "/heart/inc/main.php" и находим строку:

define( 'CACHE_TIME', '0' ); // секунд

и меняем '0' на '60', например. Т.е. контент будет кешироваться на 1 минуту.
Конечная строка будет:

define( 'CACHE_TIME', '60' ); // секунд
